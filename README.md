# 🛡️ AEC-OOP-CPP-Lab
**Comprehensive Object-Oriented Programming Portfolio | 3rd Semester CSE** *Assam Engineering College (Batch 2024-2028)*

This repository serves as a technical log of my transition from procedural C to **Object-Oriented C++**. It contains some modular programs demonstrating the "Four Pillars" of OOP, memory management, and file handling.

## 🚀 Core Technical Modules
1. **Encapsulation & Data Hiding:** Using Access Specifiers (`private`, `public`, `protected`) to secure data members.
2. **Inheritance Models:** Implementation of Single, Multiple, Multilevel, and **Virtual Base Classes** to resolve the Diamond Problem.
3. **Polymorphism:** - *Static:* Function and Operator Overloading.
   - *Dynamic:* Using **Virtual Functions** and `override` for runtime binding.
4. **Memory Management:** Efficient use of Constructors (Parameterized/Copy) and Destructors to prevent memory leaks.
5. **Generic Programming:** Utilizing **Templates** for type-independent classes and functions.
